ML Intro

Posted on 2017-02-22 | In Machine Learning

As an introduction, basic machine learning processes and categories are explained.

ML Framework

For most ML problems:

  • formulate a task as a mapping from input to output
  • formulate performance as an error measure
  1. Task - T
  2. Experience - E: what is available; how much data we could get
  3. Output - O: number/class/probability/plan

  4. Performance measure - P: error/loss function
    Classification loss
    Accuracy
    Confusion matrix: x-axis - predictions; y-axis - ground truth
    ROC curve
    Regression Loss: MAE, MSE, RMSE

  5. Feature selection - choose the presentation of input X
    !: try not to evaluate feature selection based on errors - may be resulted from the algorithm you choose
    Instead, use domain knowledge

  6. Hypotheses Space - H: all the possible solutions
    f: X -> O

  7. ML algorithm

Example:
T: predicting weather
E: historical weather data

Types

Name Data Example
Supervised Learning task driven - training data with labels classify person given face image, predicting income
Reinforcement Learning data driven - training data with reward, not final label, solve interactive problems robots, chess games, self-drive cars
Unsupervised Learning given environment - training data without label customer grouping, data compression, anomaly detection

Roadmap for Machine Learning System

Preprocessing

  • Data cleaning
  • Data augmentation: when data is not enough
  • Split dataset:
    • training set: train model
    • validation set: evaluate and select model
    • test set
  • Feature extraction
    • get useful features:
      • hand crafted features
      • learned features: e.g. CNN
    • Reduce dimensions

Training and Selecting Models

  • Training: which model/approach to use
  • Select model: validation
    • Overfitting
    • Under fitting

Cross Validation

  • K-fold cross validation: divide dataset into k folds and one for test
  • Leave-one-out cross validation: one sample for test; works for small dataset
